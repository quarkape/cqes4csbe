    │      efile.pdf  （综合素质评价文件位置）

    │      student_list_example.xlsx  （学生名单模板文件）
    │

## :bow_and_arrow: 特色功能

- 提前配置加分规则，减少繁琐的计分过程，提高科学性和效率
- 支持文件导入导出，覆盖常见功能
- 灵活构建评价指标体系，适合不同高校不同专业的培养目标
